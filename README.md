# E-marketplace

This is the front-end of a modern __e-commerce__ website crafted using __Bootstrap 4__ by me. <br> 

### The Frontend consist of :
- Landing page
- Product listing page
- Sign up page
- Login page
- Cart page
- Setting page
- Success page

### Products are listed under five categories :
- Men's Shirts
- Watches
- T-Shirts
- Sunglasses
- Shoes
